How Scanning and Anomaly Detection Work

Understand how live network scanning and anomaly detection ensures robust, real-time protection.

Step 1: Network Packet Capture

Captures live network packets using Wireshark or tcpdump. Includes details like IPs, protocol, and more.

Step 2: Preprocessing

Converts raw packets to structured format: timestamp parsing, one-hot encoding, etc., for ML readiness.

Step 3: Feature Scaling

Uses StandardScaler to normalize features. This ensures balanced model input.

Step 4: Model Prediction

The preprocessed data is fed to a trained deep learning model to detect network anomalies.

Step 5: Anomaly Detection

The system flags anomalies like DDoS, port scans, or unauthorized activity for real-time alerts.
